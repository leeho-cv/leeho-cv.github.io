Jan 2026 - Present Research Internship at AVC Lab, Jilin University

Emotion Recognition Research Based on Reinforcement Learning and Chain-of-Thought Optimization

To address hallucinated reasoning and unstable logic when audio is missing, I proposed a GRPO-based reinforcement learning training framework, explicitly segmenting episode-level reasoning chains and jointly designing acoustic and semantic scoring models to build a semantic reward mechanism.

  • Explicitly distinguished and rewarded or penalized visual speculation versus unsupported hallucination to achieve cross-modal semantic calibration
  • Introduced perturbations and hidden-state stability rewards to form a token-level integrated reward allocation strategy
  • Provided positive feedback when reasoning logic was correct even if the final result was wrong, improving model generalization
Apr 2025 - Present Provincial Student Innovation Project

Zero-shot Learning for Multi-label Software Vulnerability Detection

As the primary project lead, I conducted research on generalized zero-shot learning methods for unseen-class recognition in multi-label vulnerability detection. Built on shared semantic-space representation learning, the project constructs an integrated framework combining code semantic modeling, label semantic learning, and interpretability analysis.

  • Designed an inference-stage calibration mechanism for the GZSL setting, improving multi-label detection through bias correction and per-class threshold optimization
  • Introduced a ranking-loss strategy to improve joint detection performance across both seen and unseen classes
  • Combined attention-weight analysis with occlusion analysis for interpretability, supporting prediction rationale analysis and subsequent method extension
